新一代 Node.js REST 开发框架

初见 restify

一个 Node.js Web 服务框架，为构建语义上正确的 RESTful Web 服务而进行了优化，可以大规模应用于生产环境。Restify 优化了内省和性能，并已经在世界上一些大规模使用 Node.js 的公司中部署使用。

生产就绪

Restify 被业界一些最受尊敬的公司用于推动世界上一些大规模的 Node.js 部署

调试方便

规模化运行需要有从信号中分离噪音来追溯问题的能力。Restify 是一开始就拥有 post-mortem 调试能力。
